## 🚀 How to Run Locally

To run this portfolio on your own machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Aryan210804/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies:**
   Make sure you have Python installed, then install Flask:
   ```bash
   pip install flask
   ```

3. **Start the Flask server:**
   ```bash
   python app.py
   ```

4. **View in browser:**
   Open your browser and navigate to `http://127.0.0.1:5000`
